顔認識技術による『あっちむいてホイ！』
======

昨今、顔認識技術が向上してきたため、HTML5とJavaScriptの技術を使用し、あっちむいてホイのゲームを行います。
ユーザー側は、PCのカメラに向かって、顔を上下左右に振り、コンピュータと対戦。
カメラとブラウザだけで「こういうこともできるんだ！」と、顔認識技術の世界に触れていただきたいと思います。

＜使用技術＞
・顔認識技術
・WebRTC getUserMedia
